# Traffic Sign Recognition System

## Overview
This project focuses on recognizing and classifying traffic signs using deep learning and computer vision techniques.

## Goal
To build a CNN-based image classification model capable of identifying traffic signs from street-level images.

## Technologies Used
- Python
- TensorFlow
- Keras
- OpenCV
- CNN

## Features
- Image preprocessing
- Data augmentation
- Traffic sign classification
- CNN model training
- Performance evaluation

## Workflow
1. Dataset collection
2. Image preprocessing using OpenCV
3. CNN model development
4. Model training and evaluation
5. Traffic sign prediction

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- Confusion Matrix

## Future Improvements
- Improve classification accuracy
- Expand traffic sign dataset
- Real-time traffic sign detection
